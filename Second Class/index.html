There are three ways to use the CSS - Cascading Style Sheet

  1. Inline Styling: this is styling a specific element example, styling the H tag or P tag.The inline styling overrides the internal styling.
  2. Internal Styling: Internal styling will be placed inside the head.
  3. External Styling: You create a new file for the external styling, you can name the file anything but it must end with .css

Lest Talk About Css Selectors.

There are two types of selectors

  1. ID: The ID is a description used to differentiate between similar elements for appropriate and accurate css selection. For instance, if you have a p tag with and ID called called description, how to select instructions for such p tags in css would be to use the '#' sign with the name description next to it.That is #description
  2. CLASS: We use '.' for class selection. For example if the class name is special, you select the class in css by typing .special.

CSS BOX MODEL

There are few types of box model in css, they are;

  1. Padding: This is internal,in the sense that this only affects the space between the element and the border.
  2. Margin: This external in the sense that this affects the space between the border and other external elements.
  3. Border: This is the area in which an element is enclosed in.
